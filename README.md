# NSIPS Data Anonymizer (NSIPSデータ匿名化ツール)

本ツールは、NSIPS（薬剤システム共通仕様）形式のテキストデータに含まれる個人情報を、ランダムな文字列に置換（匿名化）するためのPythonスクリプトです。

研究や開発の現場で、実データを用いたテストを行う際に個人情報を保護することを目的としています。

## 特徴
- NSIPSデータの構造（VERヘッダー、患者情報、医療機関・医師情報）を解析し、機密性の高い項目（氏名、住所、電話番号、IDなど）のみを選択的にマスキングします。
- 用法や薬品情報など、テストに必要なデータ構造は保持されます。
- Shift-JIS (cp932) 形式のファイルに対応しています。

## 使い方

1. プログラムと同じ階層に `A_Input` フォルダと `B_Output` フォルダを作成します。
2. `A_Input` フォルダに、匿名化したいNSIPS形式のテキストファイル（*.txt）を配置します。
3. スクリプトを実行します。
```bash
   python main.py
4.B_Output フォルダに匿名化されたファイルが出力されます。
